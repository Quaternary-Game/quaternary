## Getting Started

Before getting started, please download the latest version of [Godot Engine](https://godotengine.org/). Installing [Godot from Steam](https://store.steampowered.com/app/404790/Godot_Engine/) is recommended for better control over versions.

These are the steps to get started:

1. Clone the repository
2. Open `src/project.godot`
